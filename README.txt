# tp

Il s’agit d’un programme qui fonctionne en mode console.
Une fois la compilation effectuée, un menu complet avec une riche sélection de filtres s’affichera. Chaque filtre possède un numéro attitré (il est précisé avant le nom de ce dernier).
Il suffira alors d’écrire le numéro associé au filtre que vous désirez essayer puis d’appuyer sur la touche “enter” de votre clavier.. 
Deux images s’afficheront, l’image “Originale” ainsi que l’image filtrée. L’image “Original” servira de témoin pour mettre en avant les atouts du filtre utilisé.
